# 🔧 ASP.NET Backend Automation using Batch Files

## 📌 Overview

This project demonstrates how to automate backend operations using Windows batch files.

It includes:

* Running an ASP.NET backend
* Calling REST APIs using curl
* Executing SQL stored procedures using sqlcmd

---

## ⚙️ Features

* 🚀 Start backend server automatically
* 🔗 Trigger API calls with JSON payload
* 🗄️ Execute SQL stored procedures with parameters
* 📄 Save output to file

---

## 🧱 Tech Stack

* ASP.NET Core
* SQL Server
* Batch Scripting (.bat)
* curl
* sqlcmd

---

## ▶️ How to Run

### 1. Start Backend

run:
run-backend.bat

---

### 2. Call API

run:
call-api.bat

---

### 3. Execute Stored Procedure

run:
run-procedure.bat

---

## 📘 Learning Outcomes

* Understanding of batch scripting
* Automating backend workflows
* Integrating APIs and databases
* Handling real-world debugging issues

---

## ⚠️ Notes

* Update file paths before running scripts
* Ensure .NET SDK and sqlcmd are installed
* Do not expose database credentials in production
